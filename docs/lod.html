LOD

Separator:​

  • Right-click the empty space in the left pane of MO2 and select Create separator.
  • Name the separator LOD.

If you don't use all of the terrain altering mods from the Content section, you should follow the instructions on the VNV LOD guide for generating your own LOD.

If you are using all of those mods, then you can use our pre-generated LOD by following these instructions.

all of the following mods are REQUIRED:

LOD Fixes and Improvements - NVSE​

Installation:​

  • Main File - LOD Fixes and Improvements - NVSE
  • Main File - LOD Fixes and Improvements - INI

Much Needed LOD​

Installation:​

  • Main File - Much Needed LOD
  • Delete MuchNeededLOD.esp and the meshes\effects\ folder.

Much Needed LOD fixed rocks color​

LOD additions and improvements​

Installation:​

  • Main File - LODadditions
  • Delete tmzLODadditions.esp.
  • Main File - LOD additions and improvements - TTW Patch

FNV LOD Supplementation​

Installation:​

  • Main File - FNV LOD Supplementation
  • Optional File - Optional Overpasses

TCM's LOD Overhaul​

More LODs Additions and fixes​

Installation:​

  • Main File - More LODs Additions and fixes

Wasted LOD - Cliffs of Mojave​

Installation:​

  • Delete Wasted Mojave Rock LOD.esp.

Wasted LOD - Cliffs of DC​

Different LOD mods little tweaks and additions​

LODIFY - Level of detail improvement for your Fallout (TTW And NV Lods)​

Installation:​

  • Main File - LODIFY
  • Delete TTWLods.esp.
  • Optional File - LODIFY FNV Wall_SoGB Editon
  • Delete LODIFY_FNV.esp.

Decent LOD Kit​

Installation:​

  • Main File - Decent LOD Kit
  • Delete Decent LOD Kit.esp.
